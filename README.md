
1. Цель работы
Ознакомление с методами экстраполяции, методами оценки погрешности с помощью экстраполяции, методами уточнения результатов численного эксперимента.

2. Задачи работы
Овладение методами практической оценки погрешности результатов численных экспериментов и методами уточнения результатов расчетов с помощью экстраполяции.
